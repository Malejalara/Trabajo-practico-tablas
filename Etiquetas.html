Continuación Etiquetas

Etiqueta sup

se utiliza como super indice, ej ecuaciones

Etiqueta sub

se utiliza como sub indice

Etiqueta ol y li

La etiqueta ol se utiliza para crear listas ordenadas, (items)
la etiqueta il es la que da el orden de la lista (numeración)

Ejemplo

  1. Maria
  2. Alejandra
  3. Largo
  4. Ramos

Etiqueta ul

se utiliza para crear listas desordenadas, es decir no importa el orden de los items. (viñetas)

Ejemplo

      Maria
      Alejandra
      Largo
      Ramos

ATRIBUTOS

los atributos proporcionan informacion adicional sobre el contenido de un
elemento. Ellos aparecen en la etiqueta de apertura del elemento y son
compuestos por dos partes; un mombre y un valor, separados por un signo
igual.

Atributos de alineción

centrar

para utilizar esta funcion usamos la etiqueta (p align="center")(/p)

alinear a la izquierda

para utilizar esta funcion usamos la etiqueta (p align="left")(/p)

alinerar a la derecha

para utilizar esta funcion usamos la etiqueta (p align="right")(/p)

IMAGENES

Etiqueta img y atributo src="direccion de la imagen"

permite mostar imagenes, esta etiqueta no tiene cierre

imagenes de la web

imagenes del PC

atributo width

este atributo se utiliza para definir el tamaño de la imagen y se agrega al final de la direccion

Etiqueta img y atributo src="direccion de la imagen" width="300" height="500"

width se refiere al ancho de la imagen y el nurero son los pixeles

height se refiere al alto de la imagen y el numero son los pixeles

Atributo alt="descripción o infromación

este atributo permite visualizar algun tipo de informacion sobre la imagen en caso de que no se pueda visualizar

Atributo title="descripción o infromación

este atributo permite visualizar algun tipo de informacion sobre la imagen alpasar el cursor sobre ella

Podemos combiar atributos para tener una mejor visualizacion de las imagenes, por ejemplo atributos de alineacion
